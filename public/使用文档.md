# My Portfolio 使用文档

## 概述

个人投资仪表盘，一站式管理 VGT + SMH + BTC 定投组合。六页仪表盘，支持持仓追踪、实时行情、提款规划、多设备云端同步。

**线上地址**：`https://wealth-tracker.ssp2180481336.workers.dev`

---

## 1. 六大页面

### 📊 仪表盘

- **账户现金**：记录入金/出金。买入时自动扣减，卖出时自动增加。
- **持仓明细**：展示各标的持仓数量、加权均价、现价、市值、盈亏。
- **买卖录入**：手动录入交易（日期、标的、数量、单价）。VGT、SMH、BTC 三个标的外的不记录。
- **再平衡**：两种模式——
  - **换仓**：卖出超配标的，买入欠配标的，现金不变。
  - **注资**：仅买入欠配标的（需额外资金），不卖持仓。
- **目标进度**：当前总资产 / $2,500,000，带进度条。
- **高点回撤**：各标的距 52 周高点的跌幅。

### 🗺 规划

- **收入渐进加码**：27-30 岁 $2K→$2.5K，30-35 岁 $2.5K→$3.5K，35-42 岁 $3.5K→$5K，42-47 岁 $5K+。
- **退出策略**：47 岁检查回撤 → ≤20% 全仓切换至 60%VOO+30%IEF+10%SGOV → 等恢复最长到 50 岁。
- **提款模拟**：输入总资产、提取率、组合回报率、通胀率，计算年/月提取额和耗尽时间。

## 2. 云端同步

### 配置

1. 侧边栏底部「☁️ 云端同步」
2. 线上版自动填好 URL；Token 需手动填入（与 Worker AUTH_TOKEN 一致）。
3. **Token**：`自己的token`

### 自动同步

- **上传**：录入交易、入金出金、修改参数后 2 秒自动推送到云端。
- **下载**：页面打开时自动从云端拉最新数据合并到本地。
- **手动**：「⬆ 上传」/「⬇ 下载」按钮作为备用。

### 状态指示

- 数据区标题旁的灰色圆点：上传中→橙色，成功→绿色（2 秒后恢复），失败→红色。
- 右下角弹出绿色气泡"已同步到云端"（成功）或红色气泡（失败）。

## 3. Schwab CSV 导入

### 拖拽导入

直接把嘉信理财导出的 `.csv` 文件拖到页面任意位置，出现绿色边框后松手。

### 按钮导入

侧边栏「CSV」按钮选择文件。

### 支持的格式

自动识别以下列（中英文均可）：
- Date / 日期
- Action / 方向（Buy/Sell → 自动判断买卖）
- Symbol / Ticker / 代码 / 标的
- Quantity / Shares / 数量
- Price / 价格 / 成交价

只导入 VGT、SMH、BTC，其他标的自动跳过。重复导入会产生重复记录，建议每次导入前先清空交易记录。

## 4. 数据备份

- **导出**：侧边栏「📤 导出」→ 下载 JSON 文件（含全部交易、资金、参数）。
- **导入**：侧边栏「📥 导入」→ 选择 JSON 文件恢复。
- **自动备份**：勾选后每次操作自动下载备份文件到浏览器下载目录。

> ⚠️ 浏览器 localStorage 有限制（约 5MB），长期使用建议定期导出备份并配合云端同步。

---

## 5. 操作撤销

- **清空全部交易**：确认后底部弹出可点击 Toast，5 秒内点击即可恢复所有交易。
- **清空全部资金流水**：同样支持撤销，恢复后现金余额与总资产同步还原。
- 单笔删除和清仓暂无撤销。

## 6. 移动端使用

- 同一网址手机浏览器直接打开。
- 侧边栏收在顶部，点左上角 ☰ 按钮展开。
- 表单和按钮已做触控优化（最小 40px 点击区域）。
- 表格可左右滑动查看全部列。

## 7. 暗色模式

首次打开自动跟随系统。手动切换后记忆偏好，不再跟随。侧边栏点 🌙/☀ 切换亮色/暗色主题。5 种配色可选，偏好自动记忆。

## 8. 注意事项

| | |
|---|---|
| **价格数据源** | Yahoo Finance（主，Worker 代理）+ 腾讯（备），盘前/盘后以彩色圆点标注 |
| **手动设价** | 在买卖录入区最下方可手动设置某标的价格，覆盖实时价格 |
| **重复导入 CSV** | 不会去重，同一天同一笔交易会重复记录。建议导入前先清空或用「一键清空」 |
| **中枢参数** | 年化预期 13.8%（VGT 12% + SMH 14% + BTC 18%），波动率 ~25% |
| **历史回测** | BTC 历史年化包含早期 900% 极端年份，开启后终值会极度偏高，不适合做理性预测 |
| **隐私** | 数据存于 Cloudflare D1，Token 控制访问。Token 泄露他人可读写你的数据，请勿外传 |
| **清缓存影响** | 浏览器清除缓存会丢失本地数据。云端有备份的话重新下载即可恢复 |
| **多设备冲突** | 两设备同时编辑后分别上传，后者覆盖前者。建议一台设备编辑完等待同步完成再换设备 |
| **云端免费额度** | Worker 10 万次/天、D1 5GB 存储，个人使用永久免费 |
| **workers.dev 域名** | 中国大陆部分地区可能访问不稳定，可绑定自定义域名解决 |
