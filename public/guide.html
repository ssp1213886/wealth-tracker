使用文档

My Portfolio — 个人投资仪表盘

1. 三大核心模块

💼 持仓

  • 账户现金：记录入金/出金。买入自动扣减，卖出自动增加
  • 持仓明细：各标的总持仓、加权均价、现价、市值、盈亏
  • 买卖录入：手动录入交易（日期、标的、数量、单价），仅支持 VGT / SMH / BTC
  • 再平衡：换仓模式（卖出超配买入欠配）或注资模式（仅买入欠配）
  • 目标进度：当前总资产 / $2,500,000，带进度条
  • 高点回撤：各标的距历史高点跌幅

📊 模拟

  • 蒙特卡洛路径模拟，预测终值分布
  • 中位数、P10/P90、≥$2.5M 概率
  • 增长曲线：P50 中值 + P25-P75 渐变填充 + 确定性路径 + 标普对照线
  • 历史回测（含 BTC 早期极端值，结果极度乐观，谨慎开启）
  • 灵敏度分析：+$100/月 效果、VGT 权重 +5% 效果

🗺 规划

  • 收入渐进加码：27-47 岁四阶段
  • 退出策略：47 岁按回撤条件切换防守组合，最长等到 50 岁
  • 提款模拟：输入资产/提取率/回报/通胀 → 年月提取额 + 耗尽时间

2. 云端同步

侧边栏底部「☁️ 云端同步」区域：

  • 自动上传：录入交易、入金出金后 2 秒自动推云端
  • 自动下载：页面打开时从云端拉最新数据合并
  • 状态指示：数据区灰点（上传中→橙，成功→绿，失败→红）；右下角气泡提示
  • 线上版自动填 URL 和 Token（ssp540829），本地文件需手动填
⚠ 多设备冲突：两设备同时编辑分别上传，后者覆盖前者。建议一台编辑完等同步完成再换设备。

3. Schwab CSV 导入

  • 拖拽：嘉信 CSV 文件拖到页面任意位置，绿色边框出现后松手
  • 按钮：侧边栏「CSV」按钮选择文件
  • 自动识别 Date / Symbol / Quantity / Price 列（中英文均可）
  • 仅导入 VGT / SMH / BTC，其他标的自动跳过
  • 不会去重，重复导入会产生重复记录

4. 数据备份

  • 「📤 导出」→ 下载 JSON 文件（交易 + 资金 + 参数）
  • 「📥 导入」→ 选择 JSON 文件恢复
  • 勾选「自动备份」→ 每次操作自动下载
  • 云端同步是最佳实践，本地备份作为兜底

5. 移动端

  • 手机浏览器直接打开网址
  • 侧边栏收在顶部，点左上角 ☰ 展开
  • 表单和按钮最小 40px 点击区域
  • 表格左右滑动查看全部列

6. 暗色模式

侧边栏点 🌙/☀ 切换，偏好自动记忆。

7. 注意事项

价格源腾讯行情，仅美股时段有实时数据，休市显示 --
手动设价买卖录入区下方可手动设某标的价格
模拟参数中枢年化 13.8%（VGT 12% + SMH 14% + BTC 18%）
历史回测BTC 含早期 900% 年份，结果极度乐观
隐私数据存 Cloudflare D1，Token 控制访问，请勿外传
清缓存丢失本地数据，云端有备份可下载恢复
免费额度永久免费，个人使用远达不到上限
域名部分地区可能访问不稳定，可绑定自定义域名
