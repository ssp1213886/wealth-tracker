使用文档

Wealth Tracker · 个人投资仪表盘 · 最后更新：2026-07-10

1. 页面结构

本网站包含 5 个页面，通过顶部 Tab（桌面端）和底部导航（移动端）切换：

📊 仪表盘

  • 总资产指标：账户现金、持仓市值、浮动盈亏、总盈亏
  • 📋 本月定投目标：VGT/SMH/BTC 动态金额 + 完成/待执行状态
  • 🗺 20年旅程：时间进度条（可配置起始日期和年龄）
  • 🎯 目标进度：可配置财务目标进度条
  • 📉 高点回撤：各标的距历史交易峰值跌幅
  • 📈 仓位盈亏：按标的汇总盈亏
  • 🍩 持仓分布：环形图 + 现金池标注
  • 期权持仓：活跃期权合约摘要（标的/类型/行权价/张数/到期日）

🕹️ 操作台

  • 买卖录入：手动录入交易（日期、标的、数量、价格），支持 VGT/SMH/BTC/SGOV
  • ⚖️ 年度再平衡：仅 12月31日解锁执行，其他日期可预览计算结果

期权

  • VIX 信号：实时波动率指数，影响 OTM 策略建议
  • 记录期权：录入 CALL/PUT 合约（标的、行权价、权利金、到期日、张数）
  • 盈亏汇总：本月权利金 + 累计权利金（按月写入日期统计）
  • 策略 OTM 矩阵：根据 VIX + 回撤自动计算 CALL/PUT 建议行权价百分比
  • 持仓清单：当前持有合约（含张数、到期状态、实值/虚值标记）
  • 冻结现金：PUT 卖出时行权担保金冻结，显示可用 vs 冻结

📁 数据

  • 💵 现金管理：入金/出金/修正 + 余额
  • 📊 持仓明细：标的/持仓/均价/现价/市值/盈亏
  • 📋 交易历史：全部记录 + 标的筛选
  • 💵 资金流水：出入金记录 + 修正记录

🗺 规划

  • 收入加码：四阶段年龄自动计算
  • 退出策略：47岁按回撤切换防守组合
  • 提款模拟：输入资产/提取率/回报/通胀

2. 侧边栏

  • 总资产：总额/持仓/现金 + 期权持仓摘要
  • 实时价格：VGT/SMH/BTC 实时行情
  • ⚙️ 投资参数：每月定投总额、起点日期、起始年龄、财务目标
  • 🎨 配色方案：5 种主题色（森林/海洋/暖阳/梅紫/极简）
  • 📥 数据同步：Cloudflare D1 云端 + JSON 导出 + CSV 导入

3. 云端同步

侧边栏底部「☁️ 云端同步」区域：

  • 自动上传：录入交易、入金出金后 2 秒自动推云端
  • 自动下载：页面打开时从云端拉最新数据合并
  • 状态指示：数据区灰点（上传中→橙，成功→绿，失败→红）；右下角气泡提示
  • 线上版自动填 URL，Token 需在侧栏同步面板手动填入（与 Worker 环境变量 AUTH_TOKEN 一致）
⚠ 多设备冲突：两设备同时编辑分别上传，后者覆盖前者。建议一台编辑完等同步完成再换设备。

4. Schwab CSV 导入

  • 拖拽：CSV 文件拖到页面任意位置
  • 按钮：侧边栏「CSV」按钮选择文件
  • 支持买入和卖出记录自动识别
  • 导入 VGT / SMH / BTC / SGOV

5. 数据备份

  • 「📤 导出」→ 下载 JSON 文件（交易 + 资金 + 参数），Toast 提示数量
  • 「📥 导入」→ 选择 JSON 文件恢复，Toast 提示数量
  • 勾选「自动备份」→ 每次操作自动下载

6. 现金修正

  • 数据页现金管理区域输入金额
  • 正数增加现金，负数减少现金
  • 自动记录到资金流水（修正+/修正-）
  • 适用于网站记录与实际账户不同步时的手动校准

7. 移动端

  • 手机浏览器直接打开网址
  • 底部导航 5 个 Tab 切换页面
  • 侧边栏点左上 ☰ 展开
  • 表单和按钮最小 44px 点击区域

8. 暗色模式

首次打开自动跟随系统设置。手动切换后记忆偏好，不再跟随系统。侧边栏点 🌙/☀ 切换，5 种配色可选。
