1. 页面结构

本网站包含 6 个页面，通过顶部 Tab（桌面端）和底部导航（全平台）切换：

📊 仪表盘

  • 总资产指标：账户现金、持仓市值、浮动盈亏、总盈亏
  • 🚦 异常检测：真实持仓 vs 目标比例，左显示偏离状态，右显示 SMH 信号驱动的 SGOV 备用金建议
  • 📋 本月定投目标：VGT/SMH/BTC 动态金额 + ✅完成 / ⏳待执行 状态
  • 🗺 20年旅程：时间进度条（可配置起始日期和年龄）
  • 🎯 目标进度：可配置财务目标（默认 $2.5M）进度条
  • 📉 高点回撤：各标的距历史交易峰值跌幅
  • 📈 仓位盈亏：按标的汇总 VGT/SMH/BTC 盈亏
  • 🍩 持仓分布：环形图（不含 SGOV）+ SGOV 池标注

🕹️ 操作台

  • 买卖录入：手动录入交易（日期、标的、数量、价格），支持 VGT/SMH/BTC/SGOV
  • ⚖️ 年度再平衡：仅 12月31日解锁执行，其他日期可预览计算结果

🛡️ SGOV 池

  • SGOV 余额：实时市值
  • 盾牌监测：正常监控中 / 出击信号已触发
  • SMH 信号：距 52 周高回撤 ≥ 40% 触发红色闪烁
  • SGOV 独立：不参与投资组合占比计算和再平衡

📁 数据

  • 💵 现金管理：入金/出金 + 余额
  • 📊 持仓明细：标的/持仓/均价/现价/市值/盈亏
  • 📋 交易历史：全部记录 + 标的筛选
  • 💵 资金流水：出入金记录
  • 📈 盈亏：按标的汇总盈亏（默认折叠）

📓 日志

  • 📝 观察者笔记：自由书写 + 时间倒序 + 可删除
  • 🔥 纪律打卡：24 个月定投日历 + 连续完成月数统计

🗺 规划

  • 收入加码：四阶段年龄自动计算（基于起始年龄配置）
  • 退出策略：47岁按回撤切换防守组合
  • 提款模拟：输入资产/提取率/回报/通胀

2. 侧边栏配置

  • ⚙️ 投资参数：每月定投总额、起点日期、起始年龄
  • 🎨 配色方案：5 种主题色
  • 🍩持仓占比：实时环形图
  • 💹实时价格：腾讯行情
  • 📥数据同步：Cloudflare D1 云端 + JSON 导入导出 + CSV 导入

3. 云端同步

侧边栏底部「☁️ 云端同步」区域：

  • 自动上传：录入交易、入金出金后 2 秒自动推云端
  • 自动下载：页面打开时从云端拉最新数据合并
  • 状态指示：数据区灰点（上传中→橙，成功→绿，失败→红）；右下角气泡提示
  • 线上版自动填 URL 和 Token（ssp540829），无需手动配置
⚠ 多设备冲突：两设备同时编辑分别上传，后者覆盖前者。建议一台编辑完等同步完成再换设备。

4. Schwab CSV 导入

  • 拖拽：CSV 文件拖到页面任意位置
  • 按钮：侧边栏「CSV」按钮选择文件
  • 支持买入和卖出记录自动识别
  • 导入 VGT / SMH / BTC / SGOV

5. 数据备份

  • 「📤 导出」→ 下载 JSON 文件（交易 + 资金 + 参数），Toast 提示数量
  • 「📥 导入」→ 选择 JSON 文件恢复，Toast 提示数量
  • 勾选「自动备份」→ 每次操作自动下载

6. 操作撤销

  • 清空全部交易：确认后底部弹出「🗑️ 已清空 N 笔交易 · 点击撤销」，5 秒内点击即可恢复
  • 清空全部资金流水：同样支持撤销，恢复后现金余额、数据页同步还原
  • 单笔删除和清仓操作暂无撤销（仅反馈 Toast）

7. 移动端

  • 手机浏览器直接打开网址
  • 底部导航 6 个 Tab 切换页面
  • 侧边栏点左上 ☰ 展开
  • 表单和按钮最小 40px 点击区域

8. 暗色模式

首次打开自动跟随系统设置。手动切换后记忆偏好，不再跟随系统。侧边栏点 🌙/☀ 切换，5 种配色可选。

9. 设计原则

这个网站是：20年航海日志 / 纪律打卡机 / 情绪分流器。不是：高频交易终端 / 预测分析工具 / 社交炫耀平台。

价格源腾讯行情，仅美股时段有实时数据，休市显示 --
手动设价买卖录入区下方可手动设某标的价格
模拟参数中枢年化 13.8%（VGT 12% + SMH 14% + BTC 18%）
历史回测BTC 含早期 900% 年份，结果极度乐观
隐私数据存 Cloudflare D1，Token 控制访问，请勿外传
清缓存丢失本地数据，云端有备份可下载恢复
免费额度永久免费，个人使用远达不到上限
域名部分地区可能访问不稳定，可绑定自定义域名
